= Borrar una nube
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-delete]#DELETE# [.api-doc-code-block]#`/accounts/{account_id}/topology/v1/clouds/{cloud_id}`#

Elimina la instancia de nube solicitada y todos los clústeres y volúmenes no gestionados asociados a esa nube específica.



== Parámetros

[cols="5*"]
|===
| Nombre | Tipo | En | Obligatorio | Descripción 


| account_id | cadena | ruta | Verdadero  a| 
ID del recurso de cuenta que lo contiene

* ejemplo: {{.Account}}




| cloud_id | cadena | ruta | Verdadero  a| 
ID de la colección de cloud que quieres listar

|===


== Respuesta

[source]
----
Status: 204, Indicates if the cloud resource was deleted.
----
