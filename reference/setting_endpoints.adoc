= Configuración de endpoints de Astra API (ACC)
:allow-uri-read: 


Un ajuste de cuenta es una colección de pares clave/valor que son importantes para una función conocida en una cuenta de Astra. La API de ajustes proporciona acceso a información importante sobre un ajuste de cuenta.
