= Eliminar un vínculo de rol
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-delete]#DELETE# [.api-doc-code-block]#`/accounts/{account_id}/core/v1/roleBindings/{roleBinding_id}`#

Elimina el recurso API de vinculación de funciones especificado.

*Efectos secundarios*

* La eliminación del último rol asociado a un usuario con authProvider como 'local' o 'cloud-central' (no ldap) provoca la eliminación del usuario.




== Parámetros

[cols="5*"]
|===
| Nombre | Tipo | En | Obligatorio | Descripción 


| account_id | cadena | ruta | Verdadero  a| 
ID del recurso de cuenta que lo contiene

* ejemplo: {{.Account}}




| roleBinding_id | cadena | ruta | Verdadero  a| 
ID de la colección roleBinding que se va a listar

|===


== Respuesta

[source]
----
Status: 204, Indicates if the roleBinding resource was deleted.
