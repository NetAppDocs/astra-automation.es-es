= Eliminar un grupo para un usuario
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-delete]#DELETE# [.api-doc-code-block]#`/accounts/{account_id}/core/v1/users/{user_id}/groups/{group_id}`#

Elimina el recurso API de grupo especificado.

*Efectos secundarios*

* Se eliminan los siguientes recursos asociados al grupo:
* Vinculaciones de roles que hacen referencia al grupo




== Parámetros

[cols="5*"]
|===
| Nombre | Tipo | En | Obligatorio | Descripción 


| account_id | cadena | ruta | Verdadero  a| 
ID del recurso de cuenta que lo contiene

* ejemplo: {{.Account}}




| user_id | cadena | ruta | Verdadero  a| 
ID del recurso de usuario contenedor



| grupo_id | cadena | ruta | Verdadero  a| 
ID de la colección de grupos a listar

|===


== Respuesta

[source]
----
Status: 204, Indicates if the group resource was deleted.
----
