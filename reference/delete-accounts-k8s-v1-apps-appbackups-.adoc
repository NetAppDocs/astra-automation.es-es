= Eliminar la copia de seguridad de una aplicación para una app
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-delete]#DELETE# [.api-doc-code-block]#`/accounts/{account_id}/k8s/v1/apps/{app_id}/appBackups/{appBackup_id}`#

Elimina el recurso API de copia de seguridad de aplicaciones especificado. Las copias de seguridad en curso se cancelarán y luego se eliminarán.

*Efectos secundarios*

* Si se completa una copia de seguridad, se borran los datos de la copia de seguridad.




== Parámetros

[cols="5*"]
|===
| Nombre | Tipo | En | Obligatorio | Descripción 


| account_id | cadena | ruta | Verdadero  a| 
ID del recurso de cuenta que lo contiene

* ejemplo: {{.Account}}




| app_id | cadena | ruta | Verdadero  a| 
ID del recurso que contiene la aplicación



| appBackup_id | cadena | ruta | Verdadero  a| 
ID de la colección appBackup que se va a listar

|===


== Respuesta

[source]
----
Status: 204, Indicates if the appBackup resource was deleted.
----
