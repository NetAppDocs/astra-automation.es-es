= Puntos finales de actualización de Astra API (ACC)
:allow-uri-read: 


La API de la aplicación de actualización proporciona acceso a los componentes de software susceptibles de actualización y permite iniciar actualizaciones. La disponibilidad de las actualizaciones se determina en función de los paquetes registrados en la API de paquetes.
