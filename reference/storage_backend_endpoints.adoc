= Puntos finales de almacenamiento de Astra API (ACC)
:allow-uri-read: 


La API de backend de almacenamiento proporciona acceso a representaciones de proveedores de servicios de almacenamiento (backends de almacenamiento) que pueden ser usados por clústeres y aplicaciones gestionados.
