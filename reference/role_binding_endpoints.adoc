= Puntos finales de binding de roles de Astra API (ACC)
:allow-uri-read: 


La API de vinculación de roles describe las relaciones entre los pares usuario-cuenta, o grupo-cuenta, junto con los conjuntos de permisos que se les conceden (roles).
