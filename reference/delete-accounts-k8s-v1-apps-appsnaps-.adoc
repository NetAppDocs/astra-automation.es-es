= Eliminar una instantánea de aplicación
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-delete]#DELETE# [.api-doc-code-block]#`/accounts/{account_id}/k8s/v1/apps/{app_id}/appSnaps/{appSnap_id}`#

Elimina el recurso API de Application Snapshot especificado.

*Efectos secundarios*

* Si hay una instantánea en curso, la instantánea se cancela.
* Si se completa una instantánea, los datos de la instantánea se eliminan.




== Parámetros

[cols="5*"]
|===
| Nombre | Tipo | En | Obligatorio | Descripción 


| account_id | cadena | ruta | Verdadero  a| 
ID del recurso de cuenta que lo contiene

* ejemplo: {{.Account}}




| app_id | cadena | ruta | Verdadero  a| 
ID del recurso que contiene la aplicación



| appSnap_id | cadena | ruta | Verdadero  a| 
ID de la colección appSnap que se va a listar

|===


== Respuesta

[source]
----
Status: 204, Indicates if the appSnap resource was deleted.
----
